使用mkdocs-material建站
基于python环境

1.安装mkdocs并创建项目

pip install mkdocs
pip install mkdocs-material

1.开始

选择路径
cd 你的路径
mkdocs new 文件名

2.更改主题

在vscode中打开mkdocs.yml文件进行配置
格式参考官网mkdocs

3.预览

cd 文件名
mkdocs serve
复制网址并打开（在本地服务器形成站点）有更改可随时生成并预览

2.托管在github

1.GitHub

在github创建仓库后

github仓库的推送代码↓

2.部署与构建

mkdocs gh-deploy
等待部署完成
在repository-settings-pages找到网址

3.网站优化

依旧参考官网，使用markdown语法
这里的docs>index.md是默认文章，新建md文件以增加文章
更新内容后依旧使用mkdocs gh-deploy推送到github

参考b站老师NextDeep的教程，内容尚有不足，敬请指正。
